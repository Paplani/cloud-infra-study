# Port and IP

## IP란?

IP는 네트워크에서 장치를 식별하기 위한 주소이다.

## Port란?

Port는 하나의 장치 안에서 어떤 서비스로 접속할지 구분하는 번호이다.

## 예시

- SSH: 22
- HTTP: 80
- HTTPS: 443

## HTTP vs HTTPS

### HTTP

- 포트: 80
- 암호화 ❌
- 데이터를 그대로 전송

### HTTPS

- 포트: 443
- 암호화 ✔
- SSL/TLS를 사용하여 데이터를 보호

### 차이점

HTTP는 데이터를 암호화하지 않기 때문에 보안에 취약하다.  
HTTPS는 데이터를 암호화하여 중간에서 데이터를 가로채도 내용을 확인할 수 없다.

### 실습에서의 적용

EC2 인스턴스를 생성할 때 보안 그룹에서 HTTP(80)는 열어주지만  
HTTPS(443)는 SSL 인증서 설정이 필요하기 때문에 초기 단계에서는 사용하지 않는다.
